🚀 SolveIt – Full Stack Problem Solving Platform

📌 Overview

SolveIt is a full-stack web application where users can register, log in, create problems, comment on solutions, and earn karma points. 
It includes a leaderboard system to rank users based on their activity.

.

🛠️ Tech Stack

Frontend:

React.js
React Router
Axios
Tailwind CSS

Backend:

Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication
Bcrypt.js

Deployment:

Frontend: Vercel
Backend: Render
Database: MongoDB Atlas


✨ Features

🔐 User Authentication (Register/Login)
🧑 User Profile System
📝 Create & View Problems
💬 Comment System on Problems
🏆 Karma Point System
📊 Leaderboard (Ranked by Karma)
🔒 Protected Routes (Frontend Auth Guard)
🌐 Fully Deployed Full Stack App
